Advent of Code is upon us again. I’m reminded of those paint and sip classes. You get to try something new and have fun with your friends, and both of those are great, but the end product looks a lot like everyone else’s.

I saw my friend Erik’s December Adventure, which led me to the original. I prefer Diet Mountain Dew to wine, and I’d rather paint something unique. I’m going to try a December Adventure.

Goal #

I want to continue my movement toward self-hosting:

  • Set up my own CI to replace GitHub Actions
  • Publish to my own Nginx to replace Netlify
  • Integrate archives of some old silos
  • Consider Pixelfed or Peertube for new content types

Daily log #

December 1 #

  • Extracted my Forgejo configuration, without adequately describing it.
  • Fixed an annoying issue with its ROOT_URL configuration.
  • Got interested in the SystemCrafters Live series on Forgejo (Part 1, Part 2). Saw they’ve migrated to IRC, whereupon I fell into a ZNC rabbit hole.

December 2 #

  • Set up a webserver flake module to self-host this site at a temporary subdomain.
  • Set up an ACME flake module to manage certificates.
  • Added a Forgejo identity claim to my Keyoxide profile.

December 3 #

  • Prepared to move my DNS to Hetzner DNS with OpenTofu.
  • Documented my pobox.com setup

December 4 #

  • Moved to Hetzner DNS. Mind those trailing periods!

December 5 #

  • Air travel today. Some offline cleanup tasks.
  • Set up 2FA on Forgejo. This is not a code change.
  • Set the landing page on Forgejo the “explore” view.
  • Add Forgejo contact in footer and contacts page.

December 6 #

  • Archived GitHub orgs I participate in with github2gitea-mirror.

December 7 #

  • Added support for Libravatar.

December 8 #

  • Enhanced support for Libravatar.

December 9 #

  • Automated hash calculation of Libravatar and turned it into a general NixOS module.
  • Introduced OpenTofu variables as a companion to the generalized module.
  • Created a weather station archive repo after getting that working. Self-hosting is also for weather!

December 10 #

  • Put my adventure on pause to help a friend start his hosting adventure.

December 11 #

  • Continued my friend’s adventure.

December 12 #

  • Began migration of DNS records to NixOS-DNS.

December 13 #

  • Fully implemented, but did not finish documenting, OctoDNS integration.

December 14 #

  • Finished migrating to NixOS-DNS, with deeper integration into existing modules.

December 15 #

  • Back on my friend’s adventure.

December 16 #

  • Began a refactoring of the Nix flake. No new functionality, but better organization.

December 17 #

  • Continued Nix flake refactoring. No more overlays!

December 18 #

  • Thought about rendering my Nix flake inputs as a table.
  • Abandoned experiment with a babel executor for Nix.
  • To make some progress, dropped some unused flake inputs.

December 19 #

  • A drawback to copying the Nginx configuration from the Nixpkgs Mastodon module in support of WEB_DOMAIN is that it falls out of sync. Patched the way it forwards the streaming sockets.

December 21 #

  • A day of patching and testing.

December 22 #

  • Experimenting with Opengist. I have a working Nix package, but not a NixOS module.

December 23 #

  • Published an Opengist guide with flake package and NixOS module.
  • Set up a working Opengist service at paste.rossabaker.com.

December 25 #

  • Migrated 14 years’ worth of GitHub gists to my Opengist. Gists are usually ephemeral, but revisiting old interests and hacks, and remembering some people I interacted with, brought great joy.

Retrospective #

In my adventurous December exuberance, I accomplished no original goals, but made great strides nevertheless:

  • All my code is now in a self-hosted Forgejo, as primary or a mirror.
  • All my gists are in a self-hosted Opengist.
  • All my DNS is now declared in Nix.
  • Guided a friend’s IndieWeb adventure.

I underestimated the holiday pressures. We had a late family gathering that included a day trip to Peoria, seven hours in the car plus Christmas cheer. Time well spent, but I decided to end my adventure on Christmas Day, much like the Advent of Code.

The daily log felt mostly redundant with the Git log. Next time, I’d like to write a few more brief paragraphs, even at the expense of daily bullets.

I’m glad I participated, I had fun following some others, and I was relieved by the slower pace of it all.
